Oracle Dynamic Performance Views ( also know as Fixed Views ) Catagorized

V$views are dynamic performance views based on the X$tables. Query V$FIXED_VIEW_DEFINITION for definitions of V$views. V$views are owned by SYS and are public synonyms for corresponding V_$views. V$views are cumulative in nature tracking information since startup. V$ & GV$ views are created from $ORACLE_HOME/rdbms/admin/catalog.sql. Grant access to the V$_views or GV$_views and instruct users to access the views prefaced with SYS.

Only differance between GV$ and V$ views is the instance ID column. Ensure timed_statistics = TRUE for collection of all needed information.
Replication, RAC, Streams, Heterogeneous Services, Logical Standby and Other Views here

For a complete list of the v$views use one of the three querys below:
select kqfvinam name from x$kqfvi order by kqfvinam;
select * from PUBLICSYN where sname like 'V$%' order by 3,1;
select table_name from DICTIONARY where table_name like 'V$%' order by 1;
